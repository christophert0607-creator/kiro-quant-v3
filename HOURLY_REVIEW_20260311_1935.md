### 3. US.F 空頭持倉無法平倉
**症狀**: 日誌中反覆出現 70+ 次相同錯誤：
```
⚠️ US.F 有空頭 -1 股，先平倉...
❌ 空頭平倉失敗: 您当前持有空头，请先平仓再买入多头
⛔ 風控拒絕: ShortPositionExists
```

**根因分析**:
1. 嘗試用 Market Order 平倉但可能價格為 0 導致失敗
2. 持倉同步延遲 - API 返回的持倉狀態與實際不符
3. 模擬帳戶可能對空頭平倉有特殊限制

**修復建議** (修改 `execution_engine.py`):
```python
def close_short_if_needed(self, code: str, positions: dict) -> bool:
    pos = positions.get(code, {})
    qty = pos.get("qty", 0)
    if qty < 0:
        abs_qty = abs(int(qty))
        log.warning(f"⚠️ {code} 有空頭 {qty} 股，先平倉...")
        env = ft.TrdEnv.SIMULATE if TRADE_MODE == "SIMULATE" else ft.TrdEnv.REAL
        try:
            # FIX: 使用當前市價而非 0
            current_price = self._get_current_price(code)  # 需要實現此方法
            
            ret, data = self.trade_ctx.place_order(
                price=current_price,  # 使用實際價格而非 0
                qty=abs_qty,
                code=code,
                trd_side=ft.TrdSide.BUY,
                order_type=ft.OrderType.NORMAL,  # 改用限價單
                trd_env=env,
            )
            if ret == ft.RET_OK:
                log.info(f"✅ 空頭平倉下單成功: {code} {abs_qty}股")
                time.sleep(5)  # 增加等待時間至 5 秒
                return True
            else:
                log.error(f"❌ 空頭平倉失敗: {data}")
                # FIX: 如果平倉失敗，暫時將此股票加入黑名單
                return False
        except Exception as e:
            log.error(f"❌ 平倉例外: {e}")
            return False
    return True
```

### 4. 購買力不足錯誤
**症狀**: 

**根因**: 計算倉位時使用 `NET_ASSETS = 999,998` 而非實際可用現金。  
**修復**: 在 `risk_guard.py` 中動態獲取可用現金：
```python

available_cash = acct.get("cash", 0)
max_spend = min(net * self.cfg.max_position_pct, available_cash)
```

**當前**: 順序獲取每支股票的 K 線，耗時約 3-4 秒/股。  
**優化**: 使用 asyncio 或 threading 並行獲取：
```python
from concurrent.futures import ThreadPoolExecutor

def fetch_all_klines(self) -> Dict[str, pd.DataFrame]:
    with ThreadPoolExecutor(max_workers=3) as executor:
        futures = {executor.submit(self.fetch_kline, code): code for code in STOCK_LIST}
        results = {}
        for future in futures:
            code = futures[future]
            try:
                results[code] = future.result(timeout=10)
            except Exception as e:
                log.error(f"❌ {code} 獲取失敗: {e}")
        return results
```
